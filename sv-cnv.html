STEP 7 / 14

結構變異與 CNV

超過 50 bp 的重排——刪除、重複、倒置、易位、跨基因的拷貝數改變。

Rearrangements above 50 bp — deletions, duplications, inversions, translocations, and copy-number changes.

概覽

SV / CNV 為什麼難？

SNV/Indel 在一個 read 內就能看到；但 SV（> 50 bp）通常跨越一整條 read，必須從間接訊號推斷：split reads（一條 read 兩段比對到不同位置）、discordant pairs（paired-end 距離異常）、read depth 改變（CNV）、de novo assembly（長讀長重新組裝）。

研究顯示，大多數人類功能性變異實際上是 SV/CNV，不是 SNV。但短讀長 caller 的 SV 召回率只有 30–60%；長讀長則可達 90%+。這也是 2025 年長讀長定序快速崛起的主因。

SNVs/Indels are visible within a single read; SVs (> 50 bp) typically span an entire read, so they must be inferred from indirect signals: split reads (one read aligns to two locations), discordant pairs (abnormal paired-end distance), read depth changes (CNV), de novo assembly (long-read).

Research shows most functional human variants are actually SVs/CNVs, not SNVs. But short-read SV callers achieve only 30–60% recall; long-read callers reach 90%+. This drove the long-read sequencing explosion in 2025.

💡
核心原則：不要只用一個工具！多工具 ensemble (Manta + Delly + GRIDSS) 是短讀長 SV 的 best practice。SURVIVOR / Truvari 可比較整合多個 VCF。長讀長則一個 Sniffles2 / cuteSV 通常就夠。 Core principle: Don't use a single tool! Multi-caller ensemble (Manta + Delly + GRIDSS) is best practice for short-read SVs. SURVIVOR / Truvari merge multiple VCFs. For long reads, Sniffles2 / cuteSV alone is usually enough.
SV 類型

一、五大 SV 類別

➖

DEL — Deletion

區段從基因組中刪除。訊號：覆蓋深度↓、paired insert size↑、split reads 跨 breakpoint。常見尺寸：50 bp–10 Mb。例：Duchenne 肌肉萎縮症的 DMD 外顯子缺失。

Segment removed from genome. Signals: coverage↓, paired insert size↑, split reads across breakpoints. Size: 50 bp–10 Mb. Example: DMD exon deletions in Duchenne muscular dystrophy.

➕

DUP — Duplication

區段重複插入（tandem 或 dispersed）。訊號：覆蓋深度↑、特殊 split 模式（read 的 3' 端比對到 5' 端）。例：17p11.2 的 PMP22 重複造成 Charcot-Marie-Tooth 1A。

Segment duplicated (tandem or dispersed). Signals: coverage↑, characteristic split pattern (3' end of read aligns to 5' end). Example: 17p11.2 PMP22 duplication causing CMT1A.

🔄

INV — Inversion

區段方向反轉。訊號：覆蓋深度不變、paired-end strand orientation 異常（FR → FF / RR）、split reads 跨兩個 breakpoints。例：F8 intron 22 倒置造成 ~50% 的血友病 A。

Segment flipped. Signals: coverage unchanged, abnormal paired-end strand orientation (FR → FF / RR), split reads at both breakpoints. Example: F8 intron 22 inversion causes ~50% of hemophilia A.

➡️

INS — Insertion

新序列插入基因組。挑戰：若插入長度 > read length，短讀長幾乎無法解析；長讀長能完整跨越。例：SVA / LINE-1 retrotransposon、Friedreich ataxia 的 GAA 重複擴張。

New sequence inserted into genome. Challenge: if insertion length > read length, short reads can hardly resolve; long reads span it entirely. Example: SVA/LINE-1 retrotransposons, Friedreich ataxia GAA repeat expansion.

🔁

TRA / BND — Translocation / Breakend

跨染色體的重排（或單一 breakend）。訊號：paired-end / split reads 比對到不同染色體。例：BCR-ABL t(9;22) Philadelphia 染色體（慢性骨髓性白血病）。

Inter-chromosomal rearrangement (or single breakend). Signals: paired-end / split reads mapping to different chromosomes. Example: BCR-ABL t(9;22) Philadelphia chromosome in CML.

🧮

CNV — Copy Number Variant

覆蓋深度推斷的整體拷貝數改變（0/1/2/3+）。方法：logR ratio + BAF，搭配 HMM 或 circular binary segmentation。區別：SV 是 breakpoint-resolved；CNV 通常 region-level，不知精確 breakpoint。

Coverage-based estimate of total copy number (0/1/2/3+). Method: logR ratio + BAF, with HMM or circular binary segmentation. Distinction: SVs are breakpoint-resolved; CNVs are region-level without exact breakpoints.

程式碼

SV / CNV 實作

# --- Manta (germline) ---
configManta.py \
    --bam S1.recal.bam \
    --referenceFasta ref/hg38.fa \
    --runDir manta_S1
manta_S1/runWorkflow.py -m local -j 16
# 結果 / Output: results/variants/diploidSV.vcf.gz

# --- Delly ---
delly call -t DEL -g ref/hg38.fa -o S1.delly.del.bcf S1.recal.bam
delly call -t DUP -g ref/hg38.fa -o S1.delly.dup.bcf S1.recal.bam
delly call -t INV -g ref/hg38.fa -o S1.delly.inv.bcf S1.recal.bam
delly call -t TRA -g ref/hg38.fa -o S1.delly.tra.bcf S1.recal.bam
bcftools concat -a S1.delly.*.bcf -Oz -o S1.delly.vcf.gz

# --- SURVIVOR ensemble (合併多個 caller) ---
ls manta.vcf delly.vcf gridss.vcf > vcfs.txt
SURVIVOR merge vcfs.txt 500 2 1 1 0 30 merged.sv.vcf
# 500=max bp 距離, 2=至少 2 工具支持, 1=同型別, 1=同strand, 0=disable estimate, 30=min size
# --- Sniffles2 (長讀長 SV) ---
sniffles --input ont.bam \
         --reference ref/hg38.fa \
         --vcf ont.sniffles.vcf.gz \
         --tandem-repeats ref/human_GRCh38_no_alt_analysis_set.trf.bed \
         --threads 16

# --- cuteSV (替代/ensemble) ---
cuteSV ont.bam ref/hg38.fa ont.cutesv.vcf cutesv_work/ \
   --max_cluster_bias_INS 100    --diff_ratio_merging_INS 0.3 \
   --max_cluster_bias_DEL 100    --diff_ratio_merging_DEL 0.3 \
   --threads 16 --sample S1

# --- 用 Sniffles2 做 population SV calling ---
sniffles --input S1.bam --reference ref/hg38.fa --snf S1.snf
sniffles --input S2.bam --reference ref/hg38.fa --snf S2.snf
sniffles --input S1.snf S2.snf --vcf cohort.sv.vcf.gz   # 合併
# --- CNVkit (somatic / WES) ---
cnvkit.py batch tumor.bam --normal normal.bam \
    --targets exome.bed \
    --fasta ref/hg38.fa \
    --output-reference ref.cnn \
    --output-dir cnvkit_out/ \
    -p 16
cnvkit.py scatter -s tumor.cn{s,r} -o tumor.scatter.pdf
cnvkit.py diagram -s tumor.cns -o tumor.diagram.pdf

# --- GATK gCNV (germline cohort) ---
gatk PreprocessIntervals -R ref/hg38.fa -L exome.bed --interval-merging-rule OVERLAPPING_ONLY -O intervals.interval_list
gatk CollectReadCounts -I S1.bam -L intervals.interval_list -O S1.counts.hdf5
gatk GermlineCNVCaller --run-mode COHORT -L intervals.interval_list \
    -I S1.counts.hdf5 -I S2.counts.hdf5 ... \
    --contig-ploidy-calls ploidy-calls --output-prefix cohort

# --- 註釋 AnnotSV ---
AnnotSV -SVinputFile merged.sv.vcf -outputFile merged.sv.annot.tsv \
        -genomeBuild GRCh38 -annotationsDir AnnotSV_human/
# 用 pysam 偵測 split + discordant reads（教學版）
import pysam, collections

bam = pysam.AlignmentFile("S1.bam", "rb")
sig = collections.defaultdict(list)

for r in bam.fetch("chr1", 10_000_000, 10_500_000):
    if r.is_unmapped or r.is_secondary: continue
    # split read
    if r.has_tag("SA"):
        sa = r.get_tag("SA").split(";")[0].split(",")
        sig["split"].append((r.reference_name, r.reference_start, sa[0], int(sa[1])))
    # discordant: paired but unexpected insert / orientation
    if r.is_paired and not r.mate_is_unmapped:
        if abs(r.template_length) > 1000 or r.is_reverse == r.mate_is_reverse:
            sig["discordant"].append((r.reference_name, r.reference_start, r.template_length))

print("Split reads:", len(sig["split"]))
print("Discordant pairs:", len(sig["discordant"]))
